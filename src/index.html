What is {{ site.title }} ?

{{ site.title }} is a collection of portable applications for Windows.
You can download {% gemoji inbox_tray %} your favorite application as a portable {% gemoji rocket %} setup or a 7z archive.
Each portable app is executed through an executable wrapper {% gemoji package %} [appname]-portable.exe written in Golang.

It is free and open source and always will be {% gemoji unlock %} !
