What is {{ site.title }} ?

{{ site.title }} is a collection of portable applications for Windows.
You can download :inbox_tray: your favorite application as a portable :rocket: setup or a 7z archive.
Each portable app is executed through a single EXE :package: [appname]-portable.exe written in Golang.

It is free and open source and always will be :unlock: !
To be notified of new releases you can star :star: the project on Github.
