What is {{ site.title }} ?

{{ site.title }} is a collection of portable applications for Windows. You can download {% gemoji inbox_tray %} your favorite application as a portable {% gemoji rocket %} setup or a 7z archive. Each portable app is executed through a wrapper {% gemoji package %} [appname]-portable.exe written in Golang.

It is free and open source and always will be {% gemoji unlock %}. All kinds of contributions are welcome {% gemoji raised_hands %}! The most basic way to show your support is to star {% gemoji star %} the project, or to raise issues {% gemoji speech_balloon %} You can also support this project by becoming a sponsor on GitHub {% gemoji clap %} or by making a Paypal donation to ensure this journey continues indefinitely! {% gemoji rocket %}
